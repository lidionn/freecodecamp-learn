Introduction to HTML

HTML (HyperText Markup Language) adalah bahasa markup standar yang digunakan untuk membuat dan menyusun halaman web. HTML bukan bahasa pemrograman, melainkan bahasa yang digunakan untuk menyusun elemen-elemen pada halaman web seperti teks, gambar, tautan (links), tabel, form, dan sebagainya..

Apa itu "HyperText"?

HyperText adalah teks yang memiliki tautan (link) ke teks lain. Inilah dasar dari web: dokumen yang saling terhubung melalui hyperlink.

Apa itu "Markup Language"?

Markup language digunakan untuk menandai bagian-bagian dokumen agar dapat dikenali oleh browser. Dalam HTML, markup dibuat menggunakan tag seperti

untuk paragraf atau <img> untuk gambar.
HTML bekerja bersama dengan:

  • CSS untuk mengatur tampilan (style),
  • JavaScript untuk memberikan interaktivitas.
Basic Structure of an HTML Document

Sebuah dokumen HTML terdiri atas berbagai elemen yang tersusun secara hierarkis. Berikut adalah struktur dasar: 


<!DOCTYPE html>
<html lang="en">

<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>My First HTML Page</title>
</head>

<body>
  <h1>Hello, World!</h1>
  <p>This is my first HTML page.</p>
</body>

</html>

Penjelasan:

Elemen Fungsi
<!DOCTYPE html> Menyatakan bahwa ini adalah dokumen HTML5.
<html> Elemen root dari seluruh halaman.
lang="en" Menunjukkan bahasa konten (dalam contoh ini: English).
<head> Berisi metadata, title, link stylesheet, dll. Tidak ditampilkan langsung di browser.
<meta charset="UTF-8"> Menentukan encoding karakter yang digunakan (UTF-8 mendukung hampir semua karakter).
<meta name="viewport"> Membantu halaman tampil responsif pada perangkat mobile.
<title> Judul halaman yang muncul di tab browser.
<body> Semua konten yang terlihat di browser berada di sini: teks, gambar, tombol, dll.
Common HTML Elements

HTML menyediakan berbagai elemen untuk menyusun konten. Berikut beberapa yang paling umum:

Heading (Judul)

HTML memiliki 6 level heading, dari <h1> (paling besar dan penting) sampai <h6>.


<h1>Judul Utama</h1>
<h2>Sub Judul</h2>
<h3>Judul Tingkat 3</h3>
...
<h6>Judul Tingkat 6</h6>

Gunakan heading secara hierarkis untuk membantu SEO dan keterbacaan dokumen.

Paragraph (Paragraf)

Untuk membuat paragraf, gunakan elemen <p>. Browser secara otomatis memberikan jarak antar paragraf.


<p>Ini adalah paragraf pertama.</p>
<p>Ini adalah paragraf kedua.</p>

Line Break dan Horizontal Rule

Untuk membuat jeda baris, gunakan <br>. Untuk garis horizontal, gunakan <hr>.


<p>Ini adalah paragraf pertama.<br>
Ini adalah lanjutan paragraf pertama.</p>

<hr>
  • <br> : Memasukkan baris baru tanpa membuat paragraf baru.
  • <hr> : Menambahkan garis horizontal (biasanya digunakan untuk pemisah antar bagian konten).
HTML Attributes

Setiap elemen HTML bisa memiliki atribut yang memberikan informasi tambahan. Atribut biasanya ditulis di dalam tag pembuka elemen dan selalu berbentuk pasangan nama="nilai".

Contoh Umum:


<img src="image.jpg" alt="Deskripsi gambar" width="200" height="250">

Penjelasan:

Atribut Fungsi
src Menentukan sumber gambar.
alt Memberikan deskripsi alternatif untuk gambar (penting untuk aksesibilitas).
width Menentukan lebar gambar dalam piksel.
height Menentukan tinggi gambar dalam piksel.

Atribut Global:

Beberapa atribut bisa digunakan di hampir semua elemen HTML:

  • id → Memberi identitas unik.
  • class → Digunakan untuk CSS dan JavaScript.
  • style → Menambahkan styling inline.
  • title → Teks yang muncul saat elemen di-hover.

<p id="par1" class="intro" title="Ini tooltip">Ini paragraf dengan atribut.</p>
HTML Text Formatting

HTML menyediakan berbagai tag untuk memformat teks, baik untuk penekanan, gaya, maupun fungsi tertentu.

Tag Penekanan:

Tag Fungsi Tampilan
<strong> Penekanan penting (SEO & aksesibilitas) Bold
<em> Penekanan miring (SEO & aksesibilitas) Italic
<b> Teks tebal (visual saja) Bold
<i> Teks miring (visual saja) Italic

Tag Lain:

Tag Fungsi
<mark> Menandai teks penting (highlight).
<code> Menampilkan teks sebagai kode (monospace).
<sup>/<sub> Superscript / Subscript, seperti H<sup>2</sup>O atau x<sub>2</sub>.
<blockquote> Kutipan panjang dari sumber lain.
<q> Kutipan pendek, inline.

Contoh:


<p>Ini adalah <strong>teks penting</strong> dan ini adalah <em>teks menarik</em>.</p>
<p>Air adalah H<sub>2</sub>O, dan 5<sup>2</sup> = 25.</p>
<blockquote cite="https://example.com">
  "Ini adalah kutipan panjang dari sumber tertentu."
</blockquote>
HTML Lists

HTML mendukung dua jenis list utama: Ordered List (berurutan) dan Unordered List (tak berurutan). Ada juga Description List untuk definisi/penjelasan.

Ordered List (<ol>):


<ol>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ol>

Hasilnya akan terlihat seperti ini:

  1. Item 1
  2. Item 2
  3. Item 3

Unordered List (<ul>):


<ul>
  <li>Item A</li>
  <li>Item B</li>
  <li>Item C</li>
</ul>

Hasilnya akan terlihat seperti ini:

  • Item A
  • Item B
  • Item C

Description List (<dl>):

Untuk membuat daftar istilah dan definisinya.


<dl>
  <dt>HTML</dt>
  <dd>Bahasa markup untuk membuat halaman web.</dd>
  <dt>CSS</dt>
  <dd>Bahasa untuk mendesain tampilan halaman web.</dd>
  <dt>JavaScript</dt>
  <dd>Bahasa pemrograman untuk interaktivitas halaman web.</dd
</dl>
Links & Images

Links (Tautan):

Tautan atau hyperlink adalah elemen penting dalam HTML yang menghubungkan halaman web satu sama lain atau menuju sumber eksternal.

Contoh Dasar:


<a href="https://www.example.com">Kunjungi Example</a>

Atribut Penting:

Atribut Fungsi
href Menentukan URL tujuan tautan.
target="_blank" Menentukan di mana untuk membuka tautan baru (misalnya, _blank untuk tab baru).
title Menampilkan tooltip saat kursor diarahkan
rel="noopener noreferrer" Menentukan hubungan antara dokumen saat ini dan tautan (misalnya, "noopener" untuk keamanan).

Contoh Lengkap:


<a href="https://www.example.com" target="_blank" title="Example" rel="noopener noreferrer">Example</a>

Images (Gambar):

Untuk menampilkan gambar, gunakan elemen <img>. Ini adalah tag self-closing (tidak punya tag penutup).

Contoh:


<img src="image.jpg" alt="Deskripsi gambar" width="200" height="250">

Atribut Penting:

Atribut Fungsi
src Menentukan sumber gambar.
alt Memberikan deskripsi alternatif untuk gambar (penting untuk aksesibilitas).
width/height Menentukan lebar dan tinggi gambar dalam piksel.
Catatan: Selalu gunakan atribut alt untuk gambar agar dapat diakses oleh pembaca layar dan untuk SEO. Ini juga membantu jika gambar tidak dapat dimuat.
HTML Tables

HTML menyediakan elemen untuk membuat tabel yang terdiri dari baris dan kolom. Tabel digunakan untuk menyajikan data dalam format terstruktur.

Contoh Dasar:


<table>
  <tr>
    <th>Header 1</th>
    <th>Header 2</th>
  </tr>
  <tr>
    <td>Data 1</td>
    <td>Data 2</td>
  </tr>
</table>

Penjelasan:

Tag Fungsi
<table> Membuat tabel.
<tr> Membuat baris dalam tabel.
<th> Table header (judul kolom, teks tebal dan rata tengah)
<td> Membuat sel data biasa.
<thead> Menandai bagian header tabel.
<tbody> Menandai bagian tubuh tabel.
<tfoot> Menandai bagian footer tabel.
HTML Forms

Formulir HTML digunakan untuk mengumpulkan data dari pengguna. Formulir dapat berisi berbagai elemen seperti input teks, radio button, checkbox, dropdown, dan tombol.

Contoh Dasar:


<form action="/submit" method="POST">
  <label for="name">Nama:</label>
  <input type="text" id="name" name="name">

  <input type="submit" value="Kirim">
</form>

Penjelasan:

Tag Fungsi
<form> Membuat formulir.
action URL tujuan saat formulir dikirim.
method Metode pengiriman data (GET atau POST).
<label> Membuat label untuk elemen input.
<input> Membuat elemen input (teks, radio, checkbox, dll).
Semantic HTML

Semantic HTML adalah penggunaan elemen HTML yang memiliki makna tertentu. Ini membantu mesin pencari dan pembaca layar memahami struktur dan konten halaman.

Contoh tag Semantik:

Tag Fungsi
<header> Bagian atas halaman, biasanya berisi logo dan navigasi.
<nav> Bagian navigasi, berisi tautan ke bagian lain dari halaman.
<main> Bagian utama konten halaman.
<article> Konten independen yang bisa berdiri sendiri.
<section> Bagian terpisah dalam dokumen, biasanya dengan heading.
<footer> Bagian bawah halaman, biasanya berisi informasi hak cipta dan tautan tambahan.

Penggunaan elemen semantik membantu dalam SEO dan aksesibilitas.
